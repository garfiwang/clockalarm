# Premium Timer | 精準倒數計時器

一個精美、精準且具備現代設計感的網頁倒數計時器。

👉 **線上預覽與使用**：[https://garfiwang.github.io/clockalarm/](https://garfiwang.github.io/clockalarm/)

---

## 🎨 介面與功能特色 (Features)

1. **紅白黑極簡美學**：
   - 採用**白底、黑字、紅重點色**的現代設計，並支援高對比的紅白黑深色模式。
   - 包含流暢的背景光暈與細緻的 SVG 圓形進度環倒數動畫。
2. **全螢幕沉浸計時 (Fullscreen Mode)**：
   - 一鍵切換全螢幕模式，系統會自動隱藏所有設定面板，將計時器與進度圈最大化置中，提供最無干擾的掛鐘式大螢幕計時。
3. **無誤差防休眠計時**：
   - 使用系統時間戳比對演算法，自動修正因為瀏覽器分頁進入背景、視窗最小化或系統休眠所引發的 JavaScript 計時漂移，確保倒數精準不落後。
4. **免網路相依音效合成 (Web Audio API)**：
   - 完全不需載入外部音訊檔案，防範離線或跨域（CORS）問題，利用瀏覽器原生 Web Audio API 即時合成：
     - **經典電子音 (Digital)**：逼逼逼的傳統鬧鈴。
     - **和弦風鈴 (Chime)**：多重頻率疊加的優雅風鈴聲。
     - **復古警報 (Retro)**：掃描變頻的復古電玩風警報音。
5. **瀏覽器通知**：
   - 整合瀏覽器桌面 Notification，即使瀏覽器被縮小或正在看其他分頁，時間到時亦能以系統視窗通知提醒。
6. **偏好自動儲存 (LocalStorage)**：
   - 自動記憶您上次設定的時間、選擇的音效種類、音量大小及主題設定，開箱即用。

## 🚀 快速開始 (Quick Start)

### 方式 A：直接開啟
直接在瀏覽器中雙擊並開啟 `index.html` 即可使用。

### 方式 B：本地 HTTP 伺服器
如果您想要完整體驗「瀏覽器桌面通知」功能，建議在專案目錄下啟動一個簡易伺服器：

```bash
# 使用 Python 啟動伺服器
python3 -m http.server 8000

# 或使用 Node.js 啟動伺服器
npx serve
```
接著在瀏覽器中輸入 `http://localhost:8000` (或對應埠號) 即可。
